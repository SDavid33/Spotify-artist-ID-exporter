Spotify Artist ID Exporter

Before you log in

You need your own Spotify Developer app and your own Client ID. This avoids shared app limits and lets each person use the exporter with their own Spotify setup.

Spotify Developer Dashboard:
Open Spotify Developer Dashboard
Redirect URI (hosted version):
https://spotify-artist-id-exporter-by-david33.netlify.app/callback.html
Alternative (local use):
http://localhost:8080/callback.html
Stored locally on this device
After you save your Client ID, this setup screen will disappear and only the normal exporter interface will be shown.

Quick setup

  1. Open the Spotify Developer Dashboard.
  2. Create a new app.
  3. Add the hosted or local redirect URI above.
  4. Copy the Client ID from your app page.
  5. Paste it into this exporter and continue.

Step 1 — Open the Dashboard and create your app

Step 2 — Fill the app details and add the redirect URI

Step 3 — Copy the Client ID from the app page

This version uses each user's own Spotify app. Your saved Client ID is stored only in your browser.
